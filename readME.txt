# VisitorPattern

VisitorPAttern is an exemple of an implementation and usage of the design pattern made by Dylan Rachwal for the course Advanced C++ programming at IMT Atlantique

## Usage
To compile the executable, use g++ and write in your terminal 
``` make
```
It will compile the whole classes and produce the executable
Then you can execute your program using 
``` ./main
```
